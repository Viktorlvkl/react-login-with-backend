## QUICK GUIDE TO START

First, you need to run "npm install" on the terminal from both folders.


In the terminal inside the BACKEND folder.
<br>
<b> > npm run start </b>
<br>
This starts backend.

In the terminal inside the FRONTEND folder.
<br>
<b> > npm run start </b>
<br>
This starts frontend.
